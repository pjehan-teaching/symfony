Présentation

“Symfony2 is a reusable set of standalone, decoupled, and cohesive PHP components that solve common web development problems.”

“Symfony2 is also a full-stack web framework.”

“I don't like MVC because that's not how the web works. Symfony2 is an HTTP framework; it is a Request/Response framework. That's the big deal. The fundamental principles of Symfony2 are centered around the HTTP specification.”

- Fabien Potencier

Pourquoi utiliser un framework ?

Avantages

  • Ne pas réinventer la roue
  • Architecture commune
  • Communauté importante
  • Documentation
  • Sécurité
  • Utilisation des bonnes pratiques

Inconvénients

  • Temps de chargement
  • Cadre trop strict
  • Temps d’apprentissage

Pourquoi utiliser Symfony ?

  • Nombreux Bundles
  • Nombreux projets utilisant des composants de Symfony
  • Utilisation des standards (PSR, MVC…)
  • Nombreuses offres d’emplois (surtout en France)
  • Cocorico !
